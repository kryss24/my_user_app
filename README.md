# Welcome to My Users App
***

## Task
The challenge is to do an api rest in ruby language

## Description
To build it, we use the MVC (Model-View-Controller) architecture.  
It allows user management: creation, authentication, update and deletion via HTTP routes.

## Installation
You should install the following gems:

```bash
gem install sinatra sqlite3 json httparty rspec
```

## Usage
you should do:
First: run the server
$ ruby app.rb

Then: test the endpoints with curl
$ curl -X GET -i 'http://127.0.0.1:8080/X'

Where X is the desired route (e.g., users, sign_in, etc.)
